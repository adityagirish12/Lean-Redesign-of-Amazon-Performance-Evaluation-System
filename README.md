## 💡 Why This Project Matters

Performance evaluation systems directly impact **revenue, retention, and company culture**.

This project answers a critical question:

> ❓ What happens when a $1T company evaluates performance using rankings instead of fairness?

Using Lean principles, this project uncovers **system inefficiencies, hidden bias, and process waste**, and redesigns the system into a **transparent, scalable model**.

---

## 🎯 Business Problem

Amazon’s OLR (Overall Leadership Review) system:
- Uses **forced distribution**
- Relies on **stack ranking**
- Hides actual performance ratings (OV ratings)

### 🚨 Result:
- Misaligned incentives  
- High employee stress  
- Potential talent loss

## 🔍 Key Insight

> 🔥 55% of all issues come from just TWO root causes:
- Lack of standardization  
- Poor feedback loops  

👉 Fixing these eliminates most downstream inefficiencies

## ⚠️ Current State (As-Is Process)

![Current State](./current_state.png)

### ⏱ Total Cycle Time: **~21 Weeks**

### 🚨 Bottlenecks:
- Long feedback loops  
- Rework during calibration  
- Subjective decision-making  

---

## 🧩 Root Cause Analysis

- Unfair ratings driven by **manager bias**
- Bias caused by **lack of standardized evaluation criteria**
- Criteria missing due to **unstructured feedback system**

👉 Root Cause: **System design flaw, not individual performance issue**

## 🔧 Key Improvements

### 📊 Standardization (Highest Impact)
- Structured rubrics  
- KPI-based scoring  
- Reduces manager bias  

---

### ⚙️ Remove Forced Ranking
- Shift to **absolute performance evaluation**
- Aligns incentives with actual work  

---

### 🔍 Full Transparency
- Show real OV ratings  
- Improves trust and accountability  

---

### 🤝 Coaching Over PIP
- Replace punishment with development  
- Improves retention
